Time to make a blog!

Gist

For this exercise, you need to create a blog site using sinatra.

Your site should have 4 pages:

  1. A contact me page, that should have a form
  2. A thank you page that should be shown when a user submits the form on the contact me page.
  3. A blog post page, that should have an image and some information about you.
  4. You will need to make the required templates, and a app.rb file as well.

You should add some style and flair to your site!
